# Spacemacs
1) Copy dot file:

    `cp .spacemacs ~/.spacemacs`
  
2) Run editor:

    `emacs`

# Vim
1) Set up [Vundle](https://github.com/VundleVim/Vundle.vim)

    `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
    
2) Copy dot file:
    
    `cp .vimrc ~/.vimrc`
    
3) Run vim and install plugins
    
    `vim`
    
    `:PluginInstall`
